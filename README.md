# Deloitte Data Analysis using Tableau & Excel

This repository contains data analysis tasks completed as part of a business case study for Daikibo Industrials.

Task 1 analyzes machine telemetry data using Tableau to identify which factory has the highest downtime and which device types fail most frequently.

An interactive dashboard was created with calculated fields and filters to support drill-down analysis.

Task 2 focuses on forensic analysis of employee compensation data using Excel.

Equality scores were classified into Fair, Unfair, and Highly Discriminative categories using logical formulas.

These tasks demonstrate practical problem solving using real-world datasets.

The projects highlight skills in data preparation, visualization, and business insight generation.

Tools used include Tableau and Microsoft Excel.
